Prerequisites: **git, node.js 5+, yarn**

Checkout repository
```
git clone git@github.com:topmonks/topmonks-webs.git
```

Install dependencies

```
yarn install
```

Build application to `/public` and run app on [localhost:3000](http://localhost:3000)
```
yarn run blendid
```
