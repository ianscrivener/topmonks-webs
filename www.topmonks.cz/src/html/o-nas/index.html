Jak pracujeme

Úspěchem je výsledek, ne strávený čas.

Nepotřebujeme se poflakovat v kanceláři od 9 do 5. Chceme dodat perfektní výsledek! Jelikož je náš tým většinou distribuován, usilovně jsme pracovali na tom, abychom zlepšili způsob, jakým spolupracujeme mezi sebou, a také s našimi klienty. Nepočítejte hodiny, doručte!

Slack

Efektivní komunikace je podstatou spolupráce – nejen u našich projektů, ale také u některých našich zákazníků. Je nám jedno, kde sedíte, ale jakmile pracujete, měli byste být k dispozici.

GitHub

Všechny naše projekty probíhají v GitHubu. Mnoho z nich také používá GitHub Issues pro sledování pokroku. Spolu s nástrojem Slack je alfa a omega naší spolupráce.

Quip

Zde sdílíme veškerou dokumentaci o projektech, faktury, dokonce i naši firemní knihovnu, v podstatě celé naše znalosti. Zeptejte se rarouše na jeho úžasnou kuchařku!

Swarmpit

Většina našich projektů je kontejnerizována a provozována v prostředí Docker Swarm. Pro jejich správu jsme vyvinuli vlastní uživatelské rozhraní nazvané Swarmpit.

Monksheets

Nyní se zbavte zbytečné administrace! Pokud potřebujete řídit své náklady na projekt, zažádat o volno nebo o vrácení nákladů, můžete to udělat online kdekoli a kdykoliv budete chtít.

Trello

Trello je výkonný nástroj pro sledování pokroku týmu a delegování práce.

  • 无国界 Naše firma je živý organismus provázaný s našimi partnery, zákazníky a komunitou. Nejsou zde žádné hranice.

  • 怪事 Zvláštnost, nekonvenčnost je vítána.
