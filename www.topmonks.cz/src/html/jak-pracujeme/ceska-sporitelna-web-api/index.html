Dlouho před zpracováním směrnice EU PSD2, dlouho předtím, než se veřejná API stala trendem v bankovnictví, dokonce dlouho předtím, než se takovéto nápady staly mainstreamem, náš tým už velmi dobře věděl, že veřejné brány REST API s podporou cloudových řešení byly budoucností pro banky.

Aplikace jen pro čtení? Myslete ve velkém

Když nás Česká Spořitelna v roce 2014 oslovila s nápadem vytvořit jednoduchou mobilní aplikaci, která poskytuje přístup do osobních bankovních účtů, který slouží pouze pro čtení. Přemýšleli jsme ale dál a vyvinuli průkopnickou custom-made platformu pro cloudovou integraci, která bude používaná jak interně bankou, tak externě zákazníky banky.

Srdce klientovy infrastruktury

Dnes je WebAPI srdcem celé klientovy infrastruktury. Umožňuje mu zůstat na technologické špičce jako banka s veřejným rozhraním 21. století. Většina věcí, na které pomyslíte, včetně George - nového internetového bankovnictví, běží právě na WebAPI platformě, kterou v současné době adoptovala celá Erste Group.

Na míle daleko před konkurencí

Česká spořitelna tak má obrovský náskok před svou konkurencí:

  • Podstatná část API banky byla zpřístupněná externím vývojářům pro design a testování jejich vlastních aplikací přistupujících k systému.
  • Všechny cloudové služby jsou přístupné standardizovaným způsobem (RESTfull API), který zabraňuje zmatkům a snižuje náklady při integraci.
  • WebAPI poskytuje osvědčenou, masivně škálující infrastrukturu, zvládá více než 8 miliónů požadavků denně, je podporována jednoduchým portálem pro vývojáře, vytváří platformu, která může být rekonfigurována za běhu, ihned a s nulovým prostojem.
  • Náš klient získal nástroje na správu komplexního systému a redistribuci zdrojů; možnost nastavení limitů na počet a dobu trvání požadavků, spravovat přístup k jednotlivým službám na jednom místě, sbírat záznamy (log files) pro analýzu a audit, a mnoho dalšího...
